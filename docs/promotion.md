## Reddit

### r/Python

**Title:** wealthbox-cli — open-source CLI for the Wealthbox CRM API (contacts, tasks, events, CSV export)

**Body:**

> Hey r/Python! I built an open-source CLI tool for the Wealthbox CRM API. It's aimed at financial advisors and developers who want to manage their CRM from the terminal.
>
> **What it does:**
> - Full CRUD for contacts, tasks, events, notes, and households
> - Multiple output formats (JSON, table, CSV, TSV) — pipe directly to files
> - Client-side filtering for fields the API doesn't support server-side
> - Built-in rate limiting with automatic retry
>
> **Install:** `pip install wealthbox-cli`
>
> **Built with:** Typer, httpx, Pydantic v2
>
> GitHub: https://github.com/massive-value/wealthbox-cli
> PyPI: https://pypi.org/project/wealthbox-cli/
> Docs: https://massive-value.github.io/wealthbox-cli/
>
> Feedback and contributions welcome!

### r/financialplanning or r/FinancialAdvisors

**Title:** Open-source command-line tool for Wealthbox CRM — bulk exports, scripting, automation

**Body:**

> If you use Wealthbox CRM and want to automate things like exporting contacts to CSV, bulk-updating tasks, or scripting your CRM workflows, I built an open-source CLI tool for it.
>
> `pip install wealthbox-cli` then `wbox config set-token` to get started.
>
> Examples:
> - `wbox contacts list --format csv > contacts.csv`
> - `wbox tasks list --assigned-to 123 --format table`
> - `wbox contacts add person --first-name Jane --last-name Doe --contact-type Client`
>
> GitHub: https://github.com/massive-value/wealthbox-cli
> Docs: https://massive-value.github.io/wealthbox-cli/
>
> It's free and open source (Apache 2.0). Would love feedback from other Wealthbox users.
